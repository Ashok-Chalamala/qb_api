API Endpoint Group Summary

Group Base Path Methods Description
Device Management /devices GET, POST, DELETE Register, list, and remove connected devices
Sync Operations /sync POST Trigger manual or automatic device sync
Data Ingestion /ingest POST, POST /batch Submit home tests, wearables, symptoms, SDOH
Timeline & Trends /timeline, /trends GET Retrieve unified patient timeline and trend data
Alerts /alerts GET, POST Manage clinical alerts and acknowledgments
FHIR Export /export/fhir GET, POST Export data to FHIR R4 format and push to Epic
Webhooks /webhooks POST, DELETE Register provider webhook endpoints
Analytics /analytics, /research GET Population health and research data access
Provider /provider GET Provider-specific aggregated views

Authentication & Headers

Header Value Required Used In
Authorization Bearer {jwt_token} Yes (all endpoints except webhooks) All API calls
X-API-Key {api_key} Yes (service-to-service) Ingestion endpoints
X-Request-ID {uuid} Recommended All endpoints for tracing
X-Patient-ID {patient_uuid} Yes (contextual endpoints) Timeline, alerts, export

Error Response Mapping

HTTP Status Frontend Handling API Response Example
400 Bad Request Show validation error message { "error": "schema_validation_failed", "details": ["glucose value out of range"] }
401 Unauthorized Redirect to login { "error": "token_expired", "message": "Please re-authenticate" }
404 Not Found Show "device not found" { "error": "device_not_found", "deviceId": "d1" }
429 Too Many Requests Show rate limit message + retry-after { "error": "rate_limit_exceeded", "retryAfter": 30 }
500 Internal Error Show generic error + incident ID { "error": "internal_server_error", "incidentId": "INC-12345" }

Webhook Events (Outbound)

Event Name Trigger Condition Payload Schema
critical_glucose Glucose >250 or rapid rise >50 mg/dL in 30 min { patient, value, trend, recommendation }
hypertension_trend BP >140/90 for 3 consecutive days { patient, systolic, diastolic, duration }
fall_detected Wearable fall detection trigger { patient, timestamp, location (if available) }
medication_adherence_gap Missed medication detection (2+ days) { patient, medication, daysMissed }
sleep_apnea_risk SpO2 <88% with frequent desaturations { patient, avgSpO2, desaturationEvents }
sdoh_intervention_needed High-risk SDOH response (food/housing/transport) { patient, domain, riskLevel, suggestedAction }
caregiver_concern Family observation flagged as urgent { patient, observation, suggestedFollowup }
